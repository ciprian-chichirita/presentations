  • modified - file/folder is tracked by git and got modified

  • commited - all changes got commited (snapshot)

Git: how it works
  • commit - snapshot of changes inside working directory (checksum)
  • branch - a collection of commits (you can have multiple branches with different commits)
  • tag - A tag is a mark on specific point in history
  • HEAD - is a pointer to the current branch
